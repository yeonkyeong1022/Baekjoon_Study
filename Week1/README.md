## 복잡도(Complexity)

: 알고리즘의 성능을 나타내는 척도

- 시간 복잡도: 특정한 크기의 입력에 대해 수행시간 분석
- 공간 복잡도: 특정한 크기의 입력에 대해 알고리즘의 메모리(CPU 등) 사용량 분석

**⇒ 일반적으로 복잡도가 낮을 수록 좋은 알고리즘**

### 빅오 표기법(Big-O Notation)

- 가장 빠르게 증가하는 항 만을 고려한 표기법
- 예시)
    - $f(n) = 3n^3+5n^2+1000000$ 일 때
    - $**O(N^3)$** : 차수가 가장 큰 항(지수 생략)
- 빅오에 따른 효율성

## 자료형

프로그래밍은 결국 데이터를 다루는 행위 이므로 자료형에 대한 이해가 중요

### 파이썬의 자료형

- **정수형**
    - 정수를 다루는 자료형
    - 양의 정수, 0, 음의 정수 포함
    - 코딩테스트에서 주로 정수형을 다루게 된다
    
- **실수형**
    - 소수점 아래의 데이터를 포함하는 수 자료형
    - 변수에 소수점을 붙인 수를 대입하면 실수형 변수로 처리 (4+3.2 ⇒ 실수형 변환)
    - ‘0’ 생략 가능 (***3.*** 또는 ***.123***)
    - 2진수 체계를 이용하기 때문에 정확도가 떨어진다(부동소수점)
    - `round()` 함수 : `round(소수, 반올림할 자리)` ⇒ 반올림 가능
- 복소수형
- **문자열**
    - 초기화 : 큰따옴표(`”`) 혹은 작은따옴표(`”`) 이용
    - 큰따옴표와 작은 따옴표를 구분한다.
    - 문자열 안에서 백슬래시(`\`)를 사용하여 포함
    - 문자열 덧셈(`+`) : 문자열끼리 연결해준다
    - 문자열 곱셈(`*`) : 문자열이 반복된다
    - 문자열 인덱싱, 슬라이싱 가능(특정 인덱스 값 변경 x)
    
- **리스트**
    - 여러 개의 데터를 연속적으로 담아 처리하기 위해 사용하는 자료형
    - C나 자바에서의 배열(Array)기능 및 연결리스트와 유사한 기능 지원
    - C++의 STL vector(뭐라는겨,,뭥미,,,) 와 기능적으로 유사
    - 배열 혹은 테이블이라고 부르기도 함
    - 리스트의 기능
        - 대괄호 안에 원소를 넣어 초기화, 쉼표(`,`)로 구분
        - 비어있는 리스트 선언 : `list()` or `l=[]`
        - 인덱스 값을 대괄호에 넣어 접근**(인덱싱)**
        - 대괄호 안에 콜론(`:`)을 넣어 **인덱스의 범위 지정(슬라이싱)**
        - 컴프리헨션(ex: `L = [i for i in range(10)]` )⇒ 0부터 9까지 원소 생성
            - 2차원 리스트 초기화에 유용
    
- **튜플**
    - 한 번 선언하면 변경 불가능
    - 소괄호(`()`)를 사용하여 구분
    - 공간 효율적임
    - 활용 :
        - **서로 다른 성질**의 데이터를 관리할 때
        - 데이터의 나열을 **해싱(Hashing)의 키값**으로 사용할 때
        - 메모리를 효율적으로 사용할 때
- **사전**
    - **키(Key)와 값(Value)의 쌍**을 데이터로 가지는 자료형
    - 변경 불가능한 자료(상수)형을 키로 사용 가능
    - **데이터의 조회, 수정에 있어 $O(1)$의 시간에 처리 가능**

- **집합**
    - 중복허용 x, 순서 x
    - `set()` 함수를 이용하여 초기화
    - 중괄호(`{}`)안에 콤마(`,`)로 구분하여 삽입함으로 초기화
    - $**O(1)$시간에 처리 가능**
    - 집합의 연산
        - 합집합(`|`)
        - 교집합(`&`)
        - 차집합(`-`)

### 파이썬에서 자료형의 연산

- 기본적인 사칙연산(`+, -, *`)
- 나누기연산자(`/`) : 나눠진 결과 실수형으로 반환(c의 경우 정수형으로 반환)
- 몫 연산자(`//`) : 나눠진 몫 출력(나누기 연산자의 정수형 반환과 같음)
- 나머지 연산자(`%`) : 다양한 알고리즘에 사용됨
- 거듭제곱 연산자(`**`)

### 람다 표현식

- 함수를 한 줄에 작성할 수 있음

print((lambda a, b: a+b)(3, 7))

### 자주 사용되는 라이브러리

- **내장 함수**: 기본 입출력 함수~정렬 함수까지의 기본적인 함수 제공
- **itertools** : 반복되는 형태의 데이터 처리를 위한 기능(예시 : 순열, 조합 등)
- **heapq** : 힙 자료구조 제공(최소힙)
- **bisect** : 이진 탐색(Binary Search) 기능 제공
- **collections**: 덱(Deque), 카운터(Counter) 등의 유용한 자료구조 포함
- **math** : 필수적인 수학적 기능 제공(팩토리얼, 제곱근, 최대공약수, 삼각함수, 파이 등)
